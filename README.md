# 🚀 Automated Coding Practice Tracker

An intelligent backend-driven system designed to automatically track solved coding problems and apply the SM-2 Spaced Repetition Algorithm for optimized long-term retention.

> 🐍 Python Full Stack (FastAPI + PostgreSQL)
> 📌 Status: Under Development

---

## 🎯 Project Vision

Competitive programmers often solve hundreds of problems but lack:

- A centralized tracking system
- Structured revision planning
- Intelligent memory reinforcement
- Organized documentation

This project aims to build a scalable backend system that:

✔ Fetches solved problems from coding platforms  
✔ Applies spaced repetition scheduling  
✔ Generates structured Word documentation  
✔ Tracks performance analytics  

---

## 🏗️ Planned Architecture

The system follows a server-side rendered architecture:

- **Backend:** FastAPI
- **Database:** PostgreSQL
- **ORM:** SQLAlchemy
- **Authentication:** JWT
- **Revision Engine:** SM-2 Algorithm
- **Scheduler:** APScheduler
- **Document Export:** python-docx

---

## 🧠 Core Algorithm

The project implements the SM-2 spaced repetition algorithm:

- User self-rates recall performance
- Easiness factor dynamically updated
- Review intervals automatically adjusted
- Daily revision queue generated

This ensures long-term knowledge retention instead of short-term memorization.
